# Bark & Bubbles by Ellie

Website for Bark & Bubbles by Ellie — a 1-to-1 dog grooming service in Felinfoel, Llanelli, Wales.

## Stack

- Single `index.html` with inline styles
- Tailwind CSS (CDN)
- Google Fonts (Fraunces + DM Sans)
- FormSubmit.co for contact form
- Hosted on Vercel

## Development

```bash
npm install
npm run dev        # Local server on http://localhost:3000
npm run screenshot # Puppeteer screenshots (desktop + mobile)
```

## Deployment

Pushes to `main` auto-deploy via Vercel.
